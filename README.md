#Paradigmas-programacao
> Projeto para matéria de paradigmas de programação-Q3-2014 da UFABC

##Informações
É necessário ter o compilador Chicken Scheme instalado, testado com a versão
4.7.0.

##Compilação

Servidor:
```sh
csc server.scm
```

Cliente de exemplo:
```sh
csc client.scm
```

Execute em terminais separados:
```sh
./server
```
```sh
./client
```
